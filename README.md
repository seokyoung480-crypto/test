## 소개
이 프로젝트는 **HTML, CSS, JavaScript**로 구현한 **브라우저용 테트리스 게임**입니다.  
간단하게 브라우저에서 바로 플레이할 수 있으며, 다음 기능을 제공합니다:

- 좌/우 이동, 회전, 하드 드롭, 빠른 하강
- 점수 및 최고 기록 표시 (localStorage 기반)
- 일시정지 기능 (버튼 및 `P` 키)
- 효과음 지원 (이동, 회전, 줄 제거, 게임 오버)

---

## 🕹️ 조작법

| 키 | 기능 |
|---|---|
| ⬅️ | 좌 이동 |
| ➡️ | 우 이동 |
| ⬇️ | 빠르게 내리기 |
| ⬆️ | 회전 |
| Space | 하드 드롭 |
| P | 일시정지/재개 |
| 버튼 | 일시정지/재개 |
